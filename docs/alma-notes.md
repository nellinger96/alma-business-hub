# Alma Notes

## Current Software
Alma currently uses Sentry software.

## Design Direction
Make the layout feel familiar to Sentry:
- Left sidebar
- Topbar
- Dashboard cards
- Tables
- Business workflow screens

Do not copy Sentry branding, code, or exact design.

## Important Requirement
Alma wants one app to control both businesses:
- Alianza
- Petra Insurance

Employees need their own accounts and dashboards.

Employees should be able to select a business from a dropdown if they have access to both.

Alma needs a main admin dashboard that oversees everyone.
