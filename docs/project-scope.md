# Alma Business Hub - Project Scope

## Client
Alma

## Businesses
- Alianza
- Petra Insurance

## Goal
Build one web app where Alma can manage both businesses from one hub.

## Core Requirements
- One login system
- Employees have individual accounts
- Employees can access one or both businesses based on permissions
- Dropdown to switch between Alianza and Petra Insurance
- Personalized dashboard per employee
- Alma has main admin dashboard over both businesses
- Supabase backend with RLS security
- Netlify deployment

## Included in Setup
- App shell/layout
- Business selector
- Alma admin dashboard
- Employee dashboard
- Client management
- Task/follow-up management
- Basic document tracking
- Supabase Auth
- Supabase database
- RLS policies
- Netlify deployment
